FARM label export to A2L

This configuration defines if FARM configuration data labels are exported to the A2L or not. The FARM configuration can only be viewed and merged via Calibration Data Management system if its definitions are available in the A2L file.
!Do never use available FARM labels in an application system for direct data calibration as this will result in non-predictable ECU behaviour. FARM configuration data may only be generated by the dedicated proceeding (either compilation of pre-build files within ECU SW build process or generation of executeable FARM configuration with this editor.)
