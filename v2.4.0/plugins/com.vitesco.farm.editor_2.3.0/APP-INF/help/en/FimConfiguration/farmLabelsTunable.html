FARM labels tuneable

This configuration defines if FARM labels which are exported to the A2L file are tunable or not. The FARM configuration of a SW baseline may only merged within the baseline via Calibration Data Management systems if this configuration is set.The configuration can only be set if the labels are exported to the A2L file (FARM label export to A2L)
!Do never use tunable FARM labels in an application system for direct data calibration as this will result in non-predictable ECU behaviour.FARM configuration data may only be generated by the dedicated proceeding (either compilation of pre-build files within ECU SW build process or generation of executeable FARM configuration with this editor.)
