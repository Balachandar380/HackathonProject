FIM Development Error Detection active

This configuration parameter is used to switch on or to switch off the detection of development errors during development of ECU SW. Due to implementation constraints this feature must be disabled with DEM/ERRM implementations which do not update FARM correctly (e.g. all ERRM before version 6.0.1).
Enabled: Error detection is enabled. Note that this consumes additonal RAM resources in the ECU.
Disabled: Error detection is disabled.
AUTOSAR equivalent configuration parameter : FIM_DEV_ERROR_DETECT.
