FIM Cyclic Event Evaluation

This configuration parameter specifies whether the evaluation the evaluation of DEM events is performed by the FIM cyclically or the DEM(diagnostic error management)informs FIM about changes of event states.
Enabled: Cyclic event evaluation is active.
Disabled: Cyclic event evaluation is inactive. The DEM event triggers the FIM aggregate on state changes autonomously.
AUTOSAR equivalent configuration parameter : FIM_CYCLIC_EVENT_EVALUATION
